# AGENTS.md - AI 인테리어 코디네이터

## Project Overview
AI가 분석하는 인테리어 코디네이터 - 방 사진을 업로드하면 맞춤 인테리어 시안을 생성합니다.

## Architecture
- **Frontend**: 순수 HTML/CSS/JavaScript
- **Runtime**: 브라우저
- **State Management**: 객체 기반 상태 관리
- **AI**: Mockup 시뮬레이션

## Key Files
- `index.html` - 메인 페이지
- `style.css` - 스타일링
- `script.js` - 로직

## Coding Conventions
1. **CSS**: CSS 변수 활용, BEM 네이밍
2. **JavaScript**: 모듈 패턴, 상태 관리
3. **HTML**: 시맨틱 마크업

## Workflow
Ouroboros Agent OS 워크플로우:
1. **Interview**: 요구사항 분석
2. **Seed**: 명세 생성
3. **Execute**: 구현
4. **Evaluate**: 검증
5. **Evolve**: 진화

## Agents
- **Architect**: 시스템 설계
- **Developer**: 기능 구현
- **Tester**: 검증
- **Reviewer**: 코드 리뷰

## Constraints
- 프레임워크 없음
- 모바일 반응형
- AI API 없음 (Mockup)
- 브라우저에서 바로 실행

## Success Criteria
- 모든 기능 정상 동작
- 모바일에서 사용 가능
- 로딩 시간 3초 이하
- 크로스 브라우저 호환
