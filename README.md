# Roomie — 브라우저 인테리어 코디네이터

방 사진 한 장과 생활 조건, 실제 방 치수를 바탕으로 스타일과 가구 배치를 함께 제안하는 프론트엔드 포트폴리오 프로젝트입니다. 사진은 서버로 전송하지 않고 브라우저 안에서만 처리합니다.

**[🔗 데모 사이트에서 직접 사용해보기](https://junwoo1206112.github.io/Roomie-AI-/)**

## 해결하려는 문제

인테리어 레퍼런스는 많지만 사용자는 자신의 방에 어떤 스타일이 어울리는지, 가구를 어디에 놓아야 하는지 함께 판단하기 어렵습니다. Roomie는 사진 분석부터 스타일, 실제 치수 기반 배치, 실행 계획까지 하나의 흐름으로 연결합니다.

## 핵심 기능

- JPG·PNG·WEBP 업로드 및 샘플 공간 체험
- 전체 사진 기반 밝기·채도·주요 색조 분석
- 144개 생활 조건 조합을 지원하는 결정적 추천 엔진
- 네 가지 스타일의 적합도와 선택 UI
- 실측 크기·문·창문·가구 치수를 반영한 제약 기반 배치 계산
- 충돌 없는 세 가지 배치안과 결정적 동선·공간 활용 점수
- 원본 사진 기반 컬러·조명 콘셉트 렌더링
- 실행 계획, 예산 배분, 컬러 팔레트 제공
- 콘셉트 이미지 PNG 저장
- 320px부터 대응하는 반응형 UI와 키보드 접근성

## 기술적 접근

이 프로젝트는 생성형 AI나 외부 AI API를 사용하지 않습니다. Canvas 픽셀 분석, 설명 가능한 스타일 점수, 공간 제약 기반 배치 휴리스틱을 사용한 프라이버시 중심 프로토타입입니다. 동일한 사진과 조건은 항상 동일한 결과를 반환합니다.

주요 색조는 투명 픽셀을 제외하고 RGB를 양자화한 뒤 가장 빈도가 높은 색상 군집에서 계산합니다. 사진 한 장으로 정확히 알 수 없는 면적과 생활 목적은 사용자에게 직접 입력받습니다.

배치 엔진은 25cm 단위 후보 위치를 만들고 방 경계, 출입문 여유 공간, 가구 간 간격을 통과한 후보만 평가합니다. 넓은 동선·균형 배치·벽면 집중 전략은 같은 조건에서도 서로 다른 설명 가능한 대안을 제공합니다. 기존 `공간 최적화 AI`의 고정 좌표표와 무작위 점수 방식은 사용하지 않습니다.

## 기존 공간 최적화 프로젝트 통합 범위

기존 `공간 최적화 AI`의 공간 유형, 생활 목적, 우선순위, 가구 추천, 평면 배치, 동선·활용도 지표와 인테리어 팁을 Roomie의 한 흐름에 통합했습니다. 기존 기능은 다음과 같이 확장되었습니다.

- 면적 입력과 고정 10×8 격자 → 실제 가로·세로 치수와 25cm 단위 후보 탐색
- 용도별 고정 좌표표 → 공간 유형·생활 목표별 권장 가구와 사용자가 조정할 수 있는 가구 목록
- 하나의 고정 배치 → 충돌 검사를 통과한 세 가지 고유 배치안
- 무작위 동선 점수 → 출입문 거리, 중앙 방해, 점유율을 반영한 결정적 점수
- 단순 색상 격자 → 문·창문·가구 크기를 비율대로 표시하는 평면도
- 별도 인테리어 팁 → 사진 분석, 스타일 추천, 실행 계획, 예산, 컬러 팔레트와 연결된 결과

기존의 `입체 뷰` 버튼은 화면 상태만 바꾸고 실제 3D를 렌더링하지 않았기 때문에 통합하지 않았습니다. `공부` 용도는 홈 오피스와 `집중과 생산성`, `미적 요소`는 사용자가 직접 고르는 스타일 단계로 흡수했습니다.

## 실행

Node.js 20 이상이 필요합니다. 외부 패키지는 설치하지 않습니다.

```bash
npm test
npm run build
npm start
```

`npm start`는 공개 파일만 포함한 `dist/`를 `http://127.0.0.1:4173`에서 제공합니다. 소스 루트를 그대로 정적 배포하지 마세요.

## 프로젝트 구조

```text
index.html              시맨틱 마크업
style.css               디자인 시스템과 반응형 스타일
script.js               UI 상태와 Canvas 렌더링
analysis-engine.js      테스트 가능한 순수 분석·추천 로직
layout-engine.js        치수·충돌·동선 기반 배치 최적화 로직
tests/                  Node 내장 테스트
scripts/build.js        안전한 배포 산출물 생성
scripts/serve.js        dist 전용 로컬 서버
```

## 검증 범위

- 분석 엔진 단위 테스트
- 144개 스타일 조건 조합과 배치 결정성 검사
- 방 경계·가구 충돌·출입문 여유 공간 불변식 검사
- 데스크톱·390px·320px 레이아웃 검사
- 키보드 조작과 접근성 트리 검사
- 브라우저 콘솔 오류 검사
- 개발 도구가 배포 산출물에 포함되지 않는지 검사

## 현재 한계

- 결과 이미지는 가구를 새로 합성한 생성형 이미지가 아니라 색·빛의 방향성을 보여주는 콘셉트 시안입니다.
- 문은 선택한 벽의 중앙에 있다고 가정하며 실제 위치와 기둥 등 복잡한 구조는 아직 입력할 수 없습니다.
- 추천 점수는 전문가 규칙을 단순화한 것으로 실제 시공 판단을 대체하지 않습니다.
